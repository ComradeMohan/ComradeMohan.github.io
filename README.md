> A high-performance, award-winning style personal portfolio built for technical recruiters, engineering managers, and clients. 

This portfolio is designed to showcase my skills as a Full Stack Developer & Cyber Expert, combining robust engineering with ultra-premium frontend aesthetics (micro-interactions, spring physics, and glassmorphism).

## ✨ Premium Features

- **🪄 Magnetic Buttons:** Call-to-action buttons feature physical spring-physics gravity that pulls toward the user's cursor.
- **🌊 Smooth Page Transitions:** Native app-like fluid routing with `framer-motion` cross-fades and blur reveals.
- **🔍 Command Menu (⌘K):** A developer-first global search palette for instant navigation, theme toggling, and resume downloading.
- **🖼️ Progressive Image Loading:** Custom skeleton blur-loaders that smoothly transition into high-res project mockups without layout jumping (CLS).
- **🔦 Spotlight Cards:** Beautiful glassmorphic grid cards where a radial glow tracks the exact position of your mouse.
- **🌓 Adaptive Theming:** Fully integrated Light and Dark modes with dynamic `meta` theme-color syncing.
- **📱 Fully Responsive:** Carefully crafted mobile and desktop layouts utilizing modern CSS Grid and Flexbox.

## 🛠️ Tech Stack

This project is built on a modern, blazing-fast foundation:

- **Framework:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/) (Radix Primitives)
- **Routing:** [React Router v6](https://reactrouter.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/ComradeMohan/ComradeMohan.github.io.git
cd ComradeMohan.github.io
```

### 2. Install dependencies
Ensure you have Node.js installed, then run:
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```
The site will be available at `http://localhost:5173`.

### 4. Build for Production
To generate a highly optimized production build:
```bash
npm run build
```
