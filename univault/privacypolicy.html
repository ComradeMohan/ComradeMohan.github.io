Privacy Policy — Univault

Last updated: September 26, 2025

Univault is a simple learning application created as a college project for Saveetha students. This privacy policy explains what information we collect, how we use it, and how your data is handled. The app is not connected to any university systems, LMS, or external services except Google Sign-In (optional authentication).

1. Information We Collect

1.1 Account Information

  • Name (entered manually or retrieved from Google Sign-In)
  • Email address (for login)
  • Password (stored in plain form on server since this is a student project; no encryption implemented)

1.2 Google Sign-In (Optional)

  • Name
  • Email address
  • Profile picture URL (if available)

1.3 Academic Usage Data

  • Grades added manually by the student inside the Academic Page
  • Learning mode selection (Master Mode / Pass Mode)

We DO NOT collect:

  • Course enrollments
  • Study materials or uploads
  • Groups, discussions, membership info
  • Any resource access history
  • Tags, categories, or file metadata

We also DO NOT automatically collect any technical or tracking data such as:

  • Device information
  • Android version
  • IP address or network info
  • Error logs or crash reports
  • Usage analytics, search queries, or navigation patterns

2. How We Use the Information

  • To allow Saveetha students to log in and access the learning content
  • To save learning preferences and grades entered by the user
  • To support optional Google Sign-In

No analytics, advertising, or profiling is done.

3. Data Storage & Security

This is a student project and does not use advanced security practices.

  • Passwords are stored without encryption (plain). Users should not use their real university password.
  • No sensitive logs or device identifiers are stored.
  • Only minimal account data is saved.

Since this app is for academic project purposes, it is not intended for high-security use.

4. Data Sharing

  • We do NOT share data with any university systems
  • We do NOT share data with third-party services
  • We do NOT use ads, analytics, or external trackers

Data is only used inside the app for learning purposes.

5. User Rights

  • You can delete your account by contacting the developer
  • You can modify the information inside the app
  • You may uninstall the app anytime to stop all usage

6. Age Requirements

This app is intended only for Saveetha University students aged 18+.
